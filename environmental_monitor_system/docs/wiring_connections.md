# Wiring Connections

## DHT11 Sensor
| DHT11 Pin | Arduino Pin |
|-----------|-------------|
| Data      | D4          |
| VCC       | 5V          |
| GND       | GND         |

## 16x2 I2C LCD
| LCD Pin | Arduino Pin |
|---------|-------------|
| SDA     | A4          |
| SCL     | A5          |
| VCC     | 5V          |
| GND     | GND         |

*(SDA/SCL pins may vary depending on your board — check your board's I2C pinout if not using an Uno/Nano.)*
